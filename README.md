
# Random Picker

Random Picker is a playful, modern web app for running Secret Santa, random draws, and team assignments. Add participants, assign teams, and reveal results with confetti and animations! Perfect for parties, classrooms, or any group activity where you want to let fate decide.

## Features
- 🎁 Secret Santa derangement (no one gets themselves)
- 🔀 Random winner picker
- 👥 Team assignment with dynamic team count
- 🎉 Confetti and playful animations
- 🖥️ Fullscreen reveal mode
- 💾 Local storage persistence
- 🟦 Modern, responsive UI (React + TypeScript + Vite)

## Getting Started

### 1. Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/KirthiMaharaj055/Random-Picker.git
cd Random-Picker/Random\ Picker
npm install
```

### 2. Running the App

Start the development server:

```bash
npm run dev
```

Open your browser to [http://localhost:5173](http://localhost:5173) (or the port shown in your terminal).

### 3. Software Dependencies
- Node.js (v18+ recommended)
- npm (v9+ recommended)
- Vite
- React 18
- TypeScript

### 4. Latest Release
See the [Releases](https://github.com/KirthiMaharaj055/Random-Picker/releases) page for the latest version.

## Build and Test

To build the app for production:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```

To run unit tests:

```bash
npm test
```
